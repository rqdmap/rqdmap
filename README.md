# rqdmap

This profile is intentionally minimal. It is mainly used to publish the board below.

The board is generated by [Ghostwire](https://github.com/rqdmap/ghostwire) from local **ActivityWatch** and **OpenCode** data, aggregated on a private server, and published here as a static SVG.

<picture>
  <img src="assets/dashboard.svg" alt="Personal telemetry board generated by Ghostwire" />
</picture>

Raw local data does not get published here. The public artifact is the sanitized board only.

Links: [Blog](https://www.rqdmap.top/) · [Ghostwire](https://github.com/rqdmap/ghostwire)
